# DCISMan - Pac-Man Inspired Game

#### A web-based game inspired by Pac-Man with user authentication, leaderboard, and admin panel.

<img src="https://skills.syvixor.com/api/icons?perline=15&i=java,spring,apachemaven,mysql,apache,googlecloud,cloudflare,nginx,typescript,axios,reactjs,vite,tailwindcss,bun,gsap"/>

## Tech Stack
- **Backend**: Spring Boot 3.2 (Java 17)
- **Frontend**: React 18 + Vite + Tailwind CSS
- **Database**: MySQL 8.0
- **Deployment**: Google Cloud VM (Backend + a front-end too), DCISM Server (Front-end)
